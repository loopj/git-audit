git_audit
=========

This script finds git repos with unstaged changes and/or unpushed commits (in the currently checked-out branch).

Sample Usage
------------

    git clone git://github.com/zeke/git_audit.git
    cd git_audit
    ruby git_audit.rb ~/Projects
